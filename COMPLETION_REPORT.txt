└──────────────┬───────────────────────────────────────────────┘
               │
               ▼
┌──────────────────────────────────────────────────────────────┐
│                     REDIS QUEUE (taskQueue)                  │
│              Stores job data with retry metadata             │
└──────────────┬───────────────────────────────────────────────┘
               │
               ▼
┌──────────────────────────────────────────────────────────────┐
│                  WORKER PROCESS (Node.js)                    │
│     Polls queue → Processes job → Handles retries           │
│     Success: Log completion → Remove job                    │
│     Failure: Retry (up to 3 times) → Discard               │
└──────────────┬───────────────────────────────────────────────┘
               │
               ▼
┌──────────────────────────────────────────────────────────────┐
│              BULL BOARD DASHBOARD (Monitoring)               │

                      QUICK START COMMANDS
================================================================================

PREREQUISITES:
- Node.js 18+
- Docker (for Redis)

START EVERYTHING (4 TERMINALS):

Terminal 1: Start Redis
$ docker run --name todo-redis -p 6379:6379 redis:7-alpine

Terminal 2: Start Backend API
$ cd backend
$ npm install
$ npm run migrate
$ npm run seed
$ npm run dev

Terminal 3: Start Worker
$ cd backend
$ npm run worker:dev

Terminal 4 (Optional): Start Frontend
$ cd frontend
$ npm install
$ npm run dev

VERIFY SETUP:
1. Create a todo: curl -X POST http://localhost:5000/api/todos ...
2. Check worker logs: Should see "Fake email sent"
3. Monitor jobs: http://localhost:5000/admin/queues
4. All systems go! ✅
